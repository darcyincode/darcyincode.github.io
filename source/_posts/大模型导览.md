**从基础网络到现代架构：**

- [前馈神经网络（Feedforward Neural Network）](/2026/02/04/ai-learning/大模型基础知识/前馈神经网络/) - 最基础的神经网络

## 🎯 推荐学习顺序

### 入门路径（0基础）

```
激活函数 → 前馈神经网络 → RNN → LLM学习路线图
```

**时间规划：** 约 2-4 周

### 进阶路径（有基础）

```
RNN → Transformer → 注意力机制 → LLM架构 → 预训练技术
```

**时间规划：** 约 4-8 周

### 实战路径（工程导向）

```
LLM学习路线图 → 模型微调 → 部署优化 → 应用开发
```
